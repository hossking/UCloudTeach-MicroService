# UCloudTeach-MicroService

#### Description
优云教在线教育平台微服务架构版本，使用SpringBoot+Dubbo搭建，该版本基于服务的拆分基本实现，本项目对于单体到微服务架构改造有一定的学习借鉴作用。此版架构优点是服务边界明确、各服务可独立运行、单服务维护成本较低，缺点是构建难度较大需要有相关经验。 前端项目/单体项目地址请关注Readme文件。
